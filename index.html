Welcome to the Spark hackathon!

Request your Environments

For this hackathon your team are provided with a Spark environment to run your analytics and Data Scientist Workbenches for each of the team members. We recommend that you use Data Scientist Workbench to prepare your overall project presentation.

  • Required:   As the Team Leader, Request your team's Spark Environemnt
  • Optional:   As a team member, Request your own Data Scientist Workbench

Configure Github

  • Find your teams project on HackSpark's GitHub page
  • Clone your teams repository (see instructions here)

Download the Sample Data Set

We provided a sample data set for your team, but you can also bring your own data. To load the sample data set, Login to your Spark Server and enter the following commands into the terminal:

cd /data
wget --content-disposition j.mp/BostonHackDataV2
unzip Boston-Hackathon-May-28.zip

Next you'll likely want to load the data into HDFS and run your first Spark program.
